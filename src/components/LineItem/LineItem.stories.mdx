# LineItem

**DATE ADDED**: 12/03/21 | **PROJECT**: [Quotes](https://www.notion.so/gethearth/Quotes-Bid-Estimate-Tool-c05b0b74bbb9404f8e2cc40dfdec044a)

A `LineItem` is essentially two `string` values with dots as a separator between them.
First added to support a receipt-esque run-down of values and totals in a `Quote`, with multiple line items that
add up to some total value.

<br />

---

<br />

### Usage and examples

It has props `leftContent` and `rightContent`, which are both of type `ReactNode`. It also has a
`width` prop, which is `100%` by default, and accepts a `number` or `string` so that the width can be
a percentage or absolute pixel value.

<br />

#### Default / Basic:

> The most basic usage of a `LineItem`. The default `Typography` variant used is `p1`.

<Canvas>
  <Story
    name="Basic"
    component={LineItem}
    args={{ leftContent: 'Discount', rightContent: '-$10.00' }}
  >
    {LineItemTemplate.bind()}
  </Story>
</Canvas>

#### Typography variant

> Since `leftContent` and `rightContent` both accept `ReactNode`s, we can pass in a different
> `Typography` element to render other variants or colors.

<Canvas>
  <Story
    name="Typography variant"
    args={{
      leftContent: (
        <Typography variant="h2" weight="semibold">
          Total
        </Typography>
      ),
      rightContent: (
        <Typography variant="h2" weight="semibold" color="success500">
          1,714.0
        </Typography>
      ),
    }}
  >
    {LineItemTemplate.bind()}
  </Story>
</Canvas>

#### Component variant

> We can also pass in any other component to`leftContent` and `rightContent`, such as an `icon`.

<Canvas>
  <Story
    name="Component variant"
    args={{
      leftContent: <AllInclusiveIcon fontSize="large" />,
      rightContent: <AllInclusiveIcon fontSize="large" />,
    }}
  >
    {LineItemTemplate.bind()}
  </Story>
</Canvas>
