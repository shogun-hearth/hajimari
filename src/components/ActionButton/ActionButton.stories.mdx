# ActionButton

September 6, 2022 |  https://gethearth.atlassian.net/browse/ROK-1146

The Action Button is a slightly more specific button that will be used in the Action Button Group component.
Action Buttons can be full width or smaller so that they line up horizontally.

## Action Button

`ActionButton` is built on the Button component. It has some slight variances that are specific to the Action
Button section of a workflow.
