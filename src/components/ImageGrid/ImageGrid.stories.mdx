# ImageGrid

An `ImageGrid` component accepts `children`, which should be an object array of `ReactNode`,
usually an `ImageGridItem`.

<Canvas>
  <Story
    name="Even total cards"
    args={{
      variant: 'quilted',
      rowHeight: 140,
      children: images.map((image, index) => (
        <ImageGridItem
          src={image.src}
          alt={image.alt}
          cols={index === images.length - 1 && index % 2 === 0 ? 2 : 1}
          rows={1}
        />
      )),
    }}
  >
    {ImageGridTemplate.bind({})}
  </Story>
</Canvas>
