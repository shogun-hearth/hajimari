# StatusPill

**DATE ADDED**: 11/22 | **PROJECT**: [Leads / RFQ](https://www.notion.so/gethearth/RFC-Leads-Request-a-Quote-85b916762ec841a9b126baef53366c0e)

`StatusPill` uses the `Chip` [MUI Component](https://mui.com/material-ui/react-chip/) under the hood.
Unlike most other components, we do not extend the base `ChipProps` to (a) apply a very specific set of styles
for each "variant" of the `StatusPill`, and consequently (b) avoid prop-naming collision.

## Usage

Only two props exist on `StatusPill`, both of which are required: a `label` and `variant`. A `label`
is usually some string value that will be the text contained within the pill. The `variant` can be
one of: `info`, `warning`, `error`, `success`, or `primary`. These values correspond to
different color schemes that we can use according to the context in which a `StatusPill` is used.
