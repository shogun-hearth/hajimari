export const TextFieldTemplate = (args) => {
  const [value, setValue] = useState();
  const handleChange = (event) => {
    setValue(event.target.value);
  };
  return <StatefulTextField {...args} value={value} onChange={handleChange} />;
};

# TextField

**DATE ADDED**: 12/03/21 | **PROJECT**: [Quotes](https://www.notion.so/gethearth/Quotes-Bid-Estimate-Tool-c05b0b74bbb9404f8e2cc40dfdec044a)

A `TextField` is essentially two `string` values with dots as a separator between them.
First added to support a receipt-esque run-down of values and totals in a `Quote`, with multiple line items that
add up to some total value.

<br />

---

<br />

### Usage and examples

It has props `leftContent` and `rightContent`, which are both of type `ReactNode`. It also has a
`width` prop, which is `100%` by default, and accepts a `number` or `string` so that the width can be
a percentage or absolute pixel value.

<br />
