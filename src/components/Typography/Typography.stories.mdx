# Typography

To maintain clean typography, we've opted to have a set number of allowed font-sizes and font-weights,
which can be used together in any permutation.

There are _two_ heading font-sizes and _three_ body font-sizes.
The available font-weights are _regular_, _medium_, and _semibold_, which map to _400_, _500_, and _600_ respectively.

#### Font family: Poppins

#### Font weights:

(examples below use `variant: p1`)

| Name     | Value | Example                                                                    |
| -------- | ----- | -------------------------------------------------------------------------- |
| regular  | 400   | <Typography variant="p1" weight="regular">Welcome to Hearth!</Typography>  |
| medium   | 500   | <Typography variant="p1" weight="medium">Welcome to Hearth!</Typography>   |
| semibold | 600   | <Typography variant="p1" weight="semibold">Welcome to Hearth!</Typography> |
